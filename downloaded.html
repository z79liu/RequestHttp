If you're reading this:

    If you're doing an exercise:

        Unless you did a 'GET' with '/' or '/index.html' as the path property
          (or path component of the URL parameter) intentionally:

            If so, awesome!

            If not, make sure you added the specified path to the options object you
              pass in to http.get, or append it to the URL you pass in to 'request'.

              Otherwise, you're totally missing out. (Kinda.)

    If you navigated here with your browser:

            Welcome! This is a set of example HTML pages to be retrieved with
              the Node.js http.get and 'request' modules.
